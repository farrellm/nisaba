## 2024-07-12 - Added loading spinners to authentication and settings form buttons
**Learning:** Found an opportunity to improve interaction feedback on several forms across the site (Login, Create User, Settings). I added MUI `CircularProgress` spinners to the submit buttons that appear during the async network request phase. This keeps the user informed that their action is being processed and prevents confusion over whether the click registered.
**Action:** When adding spinners, make sure they use `color="inherit"` and `size={16}` so they blend cleanly inside buttons, matching the existing button text style. Always use `mr: 1` on the `<CircularProgress>` component if text follows it within the button.
